# Design Movement: Command Topography

Command Topography is a spatial language for operational intelligence: a tension between Swiss editorial discipline and war-room urgency. The interface is built as a navigable map, not a decorative dashboard. Geometry is explicit, labels are declarative, and hierarchy is left-weighted so decision-makers can parse state quickly under pressure. The floor itself becomes a coordinate system where each desk is a functional node with identity, accountability, and visual rank.

The mood is focused and nocturnal — Navy-grounded, Surface-layered, and punctuated with surgical color signals. Gold is never ornamental: it is directional intent. Agent colors never blend into gradients; they remain discrete authorities that reveal ownership and live status. Motion appears only when something changes meaning: desk pulse on status transition, collaboration links on active handoffs, growth easing for emerging system activity.

This movement rejects template aesthetics. No centered hero tropes, no default font stack, no decorative neon. Every element must answer: does it increase operational clarity, trust, and tempo? If not, it is removed.

## Design Rules (Non-negotiable)
- **Color palette:** Navy `#0F172A`, Surface `#1A2332`, Charcoal `#1F2937`, Slate `#64748B`, Muted `#334155`, Gold `#FBBF24`, Agent accents: Bernard `#8B5CF6`, Christopher `#06B6D4`, Devan `#10B981`, Vale `#D946EF`, Scribe `#F59E0B`, Atlas `#6366F1`.
- **Typography:** Space Mono (display + labels), IBM Plex Sans (body), IBM Plex Mono (technical/code). No Inter, Roboto, or system defaults.
- **Motion:** Instant `<100ms`; micro `150ms`; reveal `200–250ms`; standard `300–400ms`; desk pulse `1.5s`; collaboration line reveal `250ms`; growth easing `cubic-bezier(0.16, 1, 0.3, 1)` for expansion moments.
- **Spacing:** 8px base grid only. Layout spacing in multiples of 8; 4px reserved for optical micro-adjustments inside components.
- **Forbidden patterns:** centered operational layouts, purple→blue gradients, rose as non-error semantic, decorative Gold usage, bounce/parallax/continuous spin, radius > 8px on cards.
- **Aesthetic DNA:** Swiss grid rationality × military command-room signaling × editorial contrast discipline.

## Unforgettable Element
The **Coordination Lattice** — animated, agent-colored collaboration lines that flash across the floor plan in 250ms bursts, making handoffs visibly traceable in real time. It turns teamwork into live cartography: the room literally lights up with cooperation.
