## Brand Essence

The Yellow Bird Company is built on the belief that intelligence should feel warm, alive, and close — not cold, distant, or mechanical. Every touchpoint should feel like a thoughtful collaborator who actually gives a damn.

**Core Promise:** We make AI feel like a teammate, not a tool.

**Tagline:** *Come Alive.*

---

## Brand Voice & Tone

### Voice Attributes
- **Direct** — We say what we mean. No corporate filler.
- **Warm** — Professional but never stiff. Human-first.
- **Playful** — We have fun around here. Wit is welcome.
- **Grounded** — No hype. The work speaks.

### Tone Modifiers by Context
| Context | Tone |
|---|---|
| Dashboard UI labels | Concise, active voice |
| Status messages | Calm and confident |
| Error states | Honest, no blame, actionable |
| Empty states | Inviting, not apologetic |
| Agent descriptions | Personality-forward |

### Do / Don't
| ✅ Do | ❌ Don't |
|---|---|
| "Agents are working" | "Processing tasks" |
| "Come alive" | "System active" |
| "Mission complete" | "Task status: done" |
| "Bernard is thinking" | "Agent 002 processing" |

## Visual Direction

### Design Principles
1. **Warmth over cold tech** — Warm neutrals, not sterile blues. The office is a real place.
2. **Motion has meaning** — Animations signal intent. An agent bouncing means they're working.
3. **Information density balanced** — Dense where needed, breathable where it matters.
4. **Brand in the details** — Yellow accents should feel like splashes of energy, not paint buckets.

### UI Surfaces
| Surface | Treatment |
|---|---|
| Dashboard background | Warm gradient: `#FAF8F5 → #F3EDE6` (light) / `#0F172A` (dark sidebar) |
| Cards | White or `#F8FAFC` with subtle warm border |
| Active states | Agent's color at 10–20% opacity fill, 30–50% border |
| Floor tiles | `#F1EDE8 / #EDE8E2` alternating |
| Round table | Radial gradient cream: `#FFFFFF → #C8B89E` |

### Border Radius Standards
- Cards: `12px` (rounded-xl)
- Badges / pills: `9999px` (rounded-full)
- Buttons: `8px` (rounded-lg)
- Scene containers: `16px`

### Shadow Standards
- Cards: `0 1px 3px rgba(0,0,0,0.06)`
- Active cards: `0 2px 12px [agent-color]22`
- Scene: `0 8px 40px rgba(0,0,0,0.08), 0 2px 8px rgba(0,0,0,0.04)`

## Logo & Identity

### Mark
- **Icon:** 🐦 Yellow Bird (canary silhouette)
- **Wordmark:** "The Yellow Bird Company" — Inter Bold, Canary Yellow on dark, Deep Slate on light
- **Short form:** "OpenClaw" — for OS/product context

### Usage Rules
- Never use the yellow bird on yellow backgrounds
- Maintain clear space equal to cap-height of wordmark on all sides
- On dark: `#F5C842` yellow on `#1E293B` slate
- On light: `#1E293B` slate on `#FAF8F5` warm white

## Motion Language

### Agent States
| State | Animation | Feel |
|---|---|---|
| Idle | Subtle opacity breathe (0.65 → 0.85) | Resting, aware |
| Active | Bounce/breathe cycle + glow ring pulse | Alive, working |
| Submitting | Float up → burst → return | Accomplishment |
| Collaborating | Dashed line pulse between desks | Connected |

### Timing Standards
- Fast (micro): 150–200ms (hover, focus)
- Standard: 300–400ms (state changes)
- Slow (ambient): 2000–3000ms (breathing, glow cycles)
- Submit arc: ~2800ms total

### Easing
- State transitions: `ease` or `cubic-bezier(0.4, 0, 0.2, 1)`
- Ambient/breathing: `ease-in-out`
- Submit arc: custom keyframe bounce
