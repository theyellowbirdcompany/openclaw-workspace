# Project: Victoria Flooring Outlet

## Project Brief
- **Client:** Victoria Flooring Outlet (victoriaflooringoutlet.ca)
- **Objective:** Produce a production-grade campaign package for the client's current Deal of the Week consisting of (1) a 1000-word blog post and (2) a dedicated landing page ready for deployment.
- **Pipeline:** Atlas → Christopher → Vale → Scribe → Devan
- **Success Criteria:**
  - Blog post is approximately 1000 words, factually grounded in the current promotion, SEO-aware, and conversion-oriented.
  - Landing page reflects the approved brand direction, highlights the approved offer framing clearly, and is implementation-ready.
  - Each downstream phase receives a clean, useful summary from the prior phase.
  - Final deliverables are production-grade and ready to deploy.
- **Constraints:**
  - Bernard approves each phase before the next begins.
  - Agents do not cross-read other agent folders.
  - PROJECT.md is the canonical shared record.
  - Claw is shadowing progress and reporting after each phase.
  - No fabricated testimonials, guarantees, or proof claims may be introduced in downstream work.

## Strategy
- Validate project structure first so all downstream work lands into the correct canonical format.
- Christopher establishes factual grounding: current deal, audience, local search opportunity, and competitive context.
- Vale converts research into a clear promotional angle, tone, messaging hierarchy, CTA strategy, and page experience guidance.
- Scribe turns approved research + brand direction into a deployable blog draft and landing page copy package.
- Devan implements the landing page and publication-ready assets using the locked direction and copy.
- Bernard reviews every phase against project objective, phase criteria, and downstream readiness before authorizing the next step.
